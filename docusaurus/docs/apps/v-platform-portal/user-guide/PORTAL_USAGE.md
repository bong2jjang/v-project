# 포털 사용법

이 문서를 읽고 나면 일반 사용자로서 포털에 로그인하고, 앱을 선택하여 작업하고, 다시 포털로 돌아오는 전체 플로우를 수행할 수 있습니다. 사이트맵 탐색, 도움말 활용, 프로덕트 투어 실행, 프로필 관리 방법도 파악할 수 있습니다.

## 로그인

1. 브라우저에서 `http://127.0.0.1:5180`을 엽니다.
2. 로그인 화면이 나타납니다.
3. 관리자가 발급한 아이디와 비밀번호를 입력하고 **로그인** 버튼을 클릭합니다. SSO가 설정된 환경이라면 SSO 버튼을 클릭하여 로그인할 수도 있습니다.
4. 로그인에 성공하면 **앱 포탈** 메인 화면으로 이동합니다.

## 앱 선택 및 접속

1. 앱 포탈 메인 화면에서 등록된 앱이 카드 형태로 나열됩니다. 각 카드에는 앱 이름, 설명, 온라인 상태, 응답 시간이 표시됩니다.
2. 접속하려는 앱 카드를 클릭합니다.
3. 새 브라우저 탭이 열리면서 해당 앱이 로딩됩니다. 포털에서 이미 로그인했으므로 앱에서 별도 로그인 화면이 나타나지 않고 바로 사용할 수 있습니다(SSO Relay).
4. 앱 카드의 오른쪽 상단에 표시된 상태 배지로 앱이 정상인지 미리 확인할 수 있습니다.
   - **Online**: 앱이 정상적으로 동작 중
   - **Degraded**: 일부 서비스에 문제가 있지만 접속은 가능
   - **Offline**: 앱에 연결할 수 없음. 관리자에게 문의하세요.

## 앱에서 작업하기

앱은 새 탭에서 열리므로, 포털 탭은 그대로 유지됩니다. 앱에서 필요한 작업을 수행합니다. 앱마다 고유한 기능과 화면이 있지만, 공통 기능(프로필, 설정, 알림 등)은 플랫폼에서 제공하므로 모든 앱에서 동일하게 사용할 수 있습니다.

## 포털로 복귀

앱에서의 작업이 끝나면 아래 방법으로 포털로 돌아올 수 있습니다.

1. 브라우저 탭 바에서 포털 탭을 클릭합니다. 포털 탭은 앱을 열 때 닫히지 않고 유지됩니다.
2. 포털 메인 화면에서 다른 앱 카드를 클릭하면 또 다른 앱에 접속할 수 있습니다.

앱 탭을 닫아도 포털 로그인 상태에는 영향이 없습니다.

## 시스템 상태 확인

앱 포탈 메인 화면을 아래로 스크롤하면 **시스템 상태** 테이블이 있습니다.

1. 모든 앱의 온라인/오프라인 상태를 한눈에 확인합니다.
2. 각 앱의 세부 서비스(데이터베이스, 캐시 등) 상태와 응답 시간이 표시됩니다.
3. 상단의 **새로고침** 버튼을 클릭하면 최신 상태를 즉시 갱신합니다. 별도 조작이 없어도 30초마다 자동 갱신됩니다.

## 사이트맵 탐색

앱 포탈 메인 화면의 하단에 **사이트맵** 섹션이 있습니다.

1. 사이트맵은 각 앱에 등록된 메뉴 항목을 앱별 카드에 모아 표시합니다.
2. 특정 기능이 어떤 앱에 있는지 찾고 싶을 때 유용합니다.
3. 메뉴 항목이 표시되지 않는 앱은 메뉴가 등록되지 않았거나 비활성 상태일 수 있습니다.

## 도움말 및 프로덕트 투어

포털의 기능과 사용법을 안내받으려면 도움말 페이지를 활용합니다.

### 도움말 페이지 접근

1. 왼쪽 사이드바에서 **도움말** 메뉴를 클릭합니다.
2. 도움말 페이지에서 포털 소개, 시작 가이드, 페이지별 기능 안내, 키보드 단축키, 자주 묻는 질문을 확인할 수 있습니다.

### 프로덕트 투어 실행

포털의 주요 기능 위치와 사용법을 화면을 따라가며 안내받을 수 있습니다.

1. 도움말 페이지에서 **전체 투어 시작** 버튼을 클릭합니다.
2. 포털 메인 화면으로 이동한 뒤 투어가 시작됩니다. 하이라이트된 영역과 설명 팝업이 순서대로 나타납니다.
3. **다음** 버튼 또는 키보드 오른쪽 화살표/Enter 키로 다음 단계로 진행합니다.
4. **이전** 버튼 또는 키보드 왼쪽 화살표/Backspace 키로 이전 단계로 돌아갑니다.
5. 투어를 중간에 종료하려면 **ESC** 키를 누릅니다.
6. 설정 페이지 전용 가이드는 도움말 페이지의 **설정 페이지 가이드** 버튼으로 시작합니다.

### 투어 초기화

한 번 완료한 투어는 다시 자동으로 표시되지 않습니다. 투어를 다시 보고 싶다면 도움말 페이지 하단의 **투어 진행 상태 초기화** 링크를 클릭합니다.

### 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| Shift + ? | 도움말 메뉴 열기 |
| Ctrl + Shift + T | 프로덕트 투어 시작 |
| ESC | 투어 진행 중 종료 |
| 오른쪽 화살표 / Enter | 투어 다음 단계 |
| 왼쪽 화살표 / Backspace | 투어 이전 단계 |

## 프로필 관리

개인 정보와 비밀번호를 관리하려면 프로필 페이지를 사용합니다.

1. 화면 상단 우측의 **사용자 아이콘**을 클릭합니다.
2. 드롭다운 메뉴에서 **프로필**을 선택합니다.
3. 프로필 페이지에서 이름, 이메일 등의 개인 정보를 확인하고 수정합니다.
4. 비밀번호를 변경하려면 왼쪽 사이드바 또는 프로필 페이지의 **비밀번호 변경** 링크를 통해 비밀번호 변경 페이지로 이동합니다.

## 알림 확인

1. 화면 상단 우측의 **알림 벨** 아이콘을 클릭합니다.
2. 알림 팝오버가 열리며 최근 알림 목록이 표시됩니다.
3. 읽지 않은 알림이 있으면 벨 아이콘에 숫자 배지가 표시됩니다.
4. 중요한 시스템 알림이나 공지사항은 페이지 상단에 배너로도 표시될 수 있습니다.

## 테마 변경

1. 화면 상단 우측의 **테마 전환 아이콘**(해/달 모양)을 클릭합니다.
2. 클릭할 때마다 라이트 모드, 다크 모드, 시스템 설정 따르기가 순환됩니다.
3. 더 세밀한 테마 설정(브랜드 색상 등)은 왼쪽 사이드바의 **설정** 메뉴에서 조정합니다.

## 자주 묻는 질문

**Q. 앱을 클릭했는데 로그인 화면이 나타나요.**
포털의 인증 토큰이 만료되었을 수 있습니다. 포털에서 로그아웃한 뒤 다시 로그인하면 새 토큰이 발급되어 앱에 자동 로그인됩니다.

**Q. 앱 상태가 Offline으로 표시돼요.**
해당 앱의 서비스가 중지되었거나 네트워크 문제가 있을 수 있습니다. 새로고침 버튼으로 상태를 다시 확인하고, 지속될 경우 관리자에게 문의하세요.

**Q. 사이트맵에 특정 앱의 메뉴가 비어 있어요.**
앱에 활성 메뉴가 등록되지 않았거나, 해당 앱이 오프라인 상태라 메뉴 정보를 가져올 수 없는 경우입니다. 관리자에게 문의하세요.

**Q. 다크 모드를 설정했는데 앱에서는 라이트 모드로 보여요.**
테마 설정은 각 앱별로 독립적으로 저장됩니다. 앱에서도 별도로 다크 모드를 설정해야 합니다.
